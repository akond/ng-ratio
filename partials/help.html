Общие сведения

Приложение «Раскладка» предназначено для составления продуктовой раскладки похода.

Для начала работы с приложением вам нужно:

  • Получить список продуктов

  • Завести информацию о планируемом походе.

  • Перейти в режим планирования похода.

Планирование

В каждый день планируемого похода (левая часть экрана) добавляйте продукты из списка в правой части экрана пока не будет достигнут запланированый уровень калорийности. Достижение этого уровня сигнализируется отображение ярлыка дня зелёным цветом, в отличие от красного, обозначающего недостаток калорий в этот день.

После того как планирование закончено можно перейти в режим отчёта закупок, где с разбиением по группам доступен весь список необходимых продуктов.
